## איך להשתמש בתבניות

1. בחרי תבנית שמתאימה למסר שרוצת להעביר
2. מלאי את הסוגריים המרובעים `[...]` בתוכן שלך
3. התאימי לשפה ולטון שלך
4. הוסיפי CTA ספציפי

---

## תבניות קרוסלה

### תבנית 1: "X טעויות"

**שקף 1 (כותרת):**
> [מספר] טעויות שבעלות עסקים עושות ב[נושא]
> (ואיך להימנע מהן)

**שקף 2:**
> את לא לבד.
> רוב ה[קהל יעד] שאני פוגשת עושות לפחות אחת מהטעויות האלה.

**שקף 3–5 (טעויות):**
> **טעות [מספר]: [שם הטעות]**
> [תיאור הטעות]
> במקום זה: [הפתרון]

**שקף אחרון:**
> שמרי את זה לכשתצטרכי
> ואם את עושה [אחת מהטעויות] — שלחי לי הודעה, נדבר

---

### תבנית 2: "המדריך ל..."

**שקף 1:**
> המדריך ל[נושא] שאף אחד לא כתב לך

**שקף 2:**
> [תיאור הבעיה — "כולם אומרים..."]
> אבל אף אחד לא מסביר [מה חסר]

**שקף 3–6 (שלבים):**
> **שלב [מספר]: [שם השלב]**
> [הסבר קצר]
> [דוגמה]

**שקף 7:**
> הכי חשוב? [תובנה מרכזית]

**שקף 8:**
> שאלות? [CTA]

---

### תבנית 3: "לפני ואחרי" (Case Study)

**שקף 1:**
> מה קורה כשבונים אסטרטגיה ל[קהל יעד]

**שקף 2:**
> הכירו את [שם/תיאור לקוחה]:
> [תיאור המצב לפני]

**שקף 3:**
> הכאב שלה:
> [ציטוט / תיאור הקושי]

**שקף 4:**
> מה עשינו:
> [3 פעולות מרכזיות]

**שקף 5:**
> התוצאות:
> [נתונים / תיאור השיפור]

**שקף 6:**
> "[ציטוט מהלקוחה]"

**שקף 7:**
> רוצה תוצאות כאלה? [CTA]

## תבניות רילס (סקריפט)

### תבנית 1: "האמת על..."

```
[0–3 שניות — וו]
"האמת על [נושא] שאף אחד לא אומר לך"

[3–15 שניות — הבעיה]
"כולם אומרים [דעה נפוצה].
אבל אחרי [ניסיון/עבודה עם X לקוחות],
גיליתי שזה לא בדיוק הסיפור."

[15–25 שניות — הפתרון]
"מה שבאמת עובד זה:
1. [נקודה ראשונה]
2. [נקודה שנייה]
3. [נקודה שלישית]"

[25–30 שניות — CTA]
"שמרי את זה ותפתחי כשתצטרכי.
ואם רוצה לדעת איך ליישם — [CTA]"
```

---

### תבנית 2: "יום בחיים"

```
[0–3 שניות]
"יום בחיים של [תפקיד] — הגרסה האמיתית"

[3–20 שניות — הצצה]
מראות של: [פעולה 1], [פעולה 2], [פעולה 3]
"מה שרוב האנשים לא רואים זה ש[תובנה]"

[20–28 שניות — ערך]
"הדבר הכי חשוב שלמדתי זה: [תובנה מרכזית]"

[28–30 שניות]
"לעוד כזה — עקבי ושמרי"
```

## תבניות כיתוב

### כיתוב לפוסט חינוכי

```
[וו — שאלה או הצהרה מפתיעה]

[פסקה 1 — הקשר והבעיה]
[2–3 משפטים]

[פסקה 2 — הפתרון/הטיפ]
[2–3 משפטים]

[פסקה 3 — דוגמה / הוכחה]
[1–2 משפטים]

[CTA]
שמרי | שתפי עם מי שצריכה | שלחי לי [מילת מפתח]

[האשטגים]
#[נישה] #[קהל יעד] #[נושא]
```

---

### כיתוב לפוסט מכירה

```
[וו — התוצאה שהלקוחה רוצה]

[בעיה — "אם את מרגישה ש..."]

[פתרון — "בדיוק בשביל זה בניתי..."]

[מה כלול]
• [פריט 1]
• [פריט 2]  
• [פריט 3]

[הוכחה — "לקוחות שאיתי..."]

[דחיפות / הגבלה — אם רלוונטי]

[CTA ברור]
"לפרטים ותיאום — שלחי לי את המילה [מילה] בהודעה"
```

## תבניות סטוריז

### סקר מעורבות
```
סטורי 1: [שאלה] + סקר "כן / לא" או "A / B"
סטורי 2: "הרוב ענו [X] — ולא לגמרי הפתיע אותי"
           [תובנה קצרה]
סטורי 3: "רוצה לשמוע עוד על זה? שלחי לי הודעה"
```

### מאחורי הקלעים — עבודה עם לקוחה
```
סטורי 1: "היום עבדנו על [X] עם אחת הלקוחות שלי"
           [תמונה / סרטון מהתהליך]
סטורי 2: "הדבר שגילינו: [תובנה]"
סטורי 3: "התוצאה? [מה קרה]"
סטורי 4: "אם גם את רוצה כזה — [CTA]"
```

```

```

