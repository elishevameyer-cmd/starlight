## 5 עמודי תוכן

כל חשבון אינסטגרם עסקי צריך 5 סוגי תוכן שמתחלפים. אם חסר אחד — הפיד נראה חד-ממדי.

| עמוד | מטרה | דוגמה |
|------|-------|--------|
| **1. חינוך** | בניית מומחיות ואמון | "3 טעויות שבעלות עסקים עושות באינסטגרם" |
| **2. השראה** | חיבור רגשי והנעה | "למה החלטתי להפסיק לפחד ולהתחיל לפרסם" |
| **3. מאחורי הקלעים** | אותנטיות וקרבה | "יום עבודה שלי כמנהלת מדיה חברתית" |
| **4. הוכחה חברתית** | ביטחון ואמינות | "מה קרה אחרי שבנינו אסטרטגיה ל-[שם לקוחה]" |
| **5. מכירה** | המרה ישירה | "שיחת RESET — בואי נבנה את האסטרטגיה שלך" |

**יחס מומלץ:** 3:1:1 (3 חינוך/השראה/BTS לכל פוסט מכירה אחד)

---

## מבנה פוסט שמוכר

### הנוסחה
```
1. וו (Hook)         — שורה ראשונה שגורמת לעצור את הגלילה
2. הקשר / בעיה      — "את מכירה את זה?" "זה קורה לך?"
3. פתרון / ערך      — הידע, הטיפ, התובנה
4. הוכחה            — דוגמה, תוצאה, סיפור קצר
5. CTA              — מה לעשות עכשיו (אחד בלבד!)
```

### דוגמה מלאה
> **וו:** "למה בעלות עסקים עוזבות אינסטגרם אחרי 3 חודשים"
>
> הן לא עצלות ולא חסרות מוטיבציה — הן פשוט מפרסמות בלי אסטרטגיה.
>
> כשאין אסטרטגיה: כל פוסט הוא "יורה ומקווה". כשיש אסטרטגיה: כל פוסט עושה עבודה.
>
> עם לקוחה אחת שלי — אחרי 6 שבועות עם אסטרטגיה ברורה — עברנו מ-0 פניות לחודש ל-8.
>
> **רוצה לדעת איך? שמרי את הפוסט הזה ↓**

## נוסחת קרוסלה (5–8 שקפים)

### מבנה
| שקף | תוכן | מטרה |
|-----|------|-------|
| **1** | כותרת חזקה + וו | לגרום ללחוץ |
| **2** | הבעיה — "את מכירה את זה?" | הזדהות |
| **3–5** | הפתרון / הטיפים (1 לשקף) | ערך |
| **6** | סיכום + הרחבה | בניית סמכות |
| **7** | CTA + הנעה לפעולה | המרה |
| **8** (אופציונלי) | עקבי לעוד תוכן כזה | צמיחה |

### כותרות שעובדות לקרוסלות
- "X דברים ש[קהל יעד] לא יודעות על [נושא]"
- "למה [פעולה] לא עובד לך (ואיך לתקן)"
- "המדריך ל[נושא] בלי [כאב נפוץ]"
- "X טעויות ש[קהל יעד] עושות — ואיך להימנע"

## נוסחת רילס

### מבנה לפי שניות
```
0–3 שניות:   וו חזק — שאלה / עובדה מפתיעה / הבטחה
3–15 שניות:  ערך / פתרון / תהליך (מהיר!)
15–25 שניות: דוגמה / הוכחה
25–30 שניות: CTA + קריאה לשמירה/עקיבה
```

### סוגי וו לרילסים
- "אף אחד לא אומר לך את זה, אבל..." (חשיפת סוד)
- "תעשי את זה ב-30 שניות ותראי שינוי" (הבטחת מהירות)
- "לכל מי שמפרסמת ולא מקבלת פניות" (ייחוד קהל)
- מראה ישיר למצלמה + שאלה פתוחה (חיבור אישי)

## נוסחת סטוריז — רצף של 3–5

### רצף מעורבות
```
סטורי 1: שאלה / סקר ("את יותר..."  /  "כן / לא")
סטורי 2: תוצאות + תגובה + ערך קצר
סטורי 3: CTA — "לפרטים נוספים שלחי לי הודעה"
```

### רצף BTS (מאחורי הקלעים)
```
סטורי 1: "היום אני עובדת על..."  (הצצה)
סטורי 2: תהליך / מחשבה / אתגר
סטורי 3: תוצאה / תובנה
סטורי 4: שאלה לקהל
```

## לוח תוכן שבועי — בסיס

| יום | סוג תוכן | פורמט |
|-----|----------|--------|
| **ראשון** | חינוך / טיפ | קרוסלה |
| **שלישי** | מאחורי הקלעים | רילס / סטוריז |
| **חמישי** | הוכחה חברתית | פוסט + כיתוב |
| **שישי** | השראה / אישי | תמונה + כיתוב |
| **יומי** | סטוריז | 2–4 סטוריז |

**הערה:** עדיף 3 פוסטים קבועים בשבוע מ-7 פוסטים שבוע אחד ואפס בשבוע הבא.

## ניתוח תוכן שעובד

### שאלות לשאול לפני כל פוסט
1. מי הקהל הספציפי שאני מדברת אליו?
2. מה הכאב / הצורך שאני פותרת?
3. מה הם יוצאים עם זה?
4. מה אני רוצה שיעשו?
5. למה עכשיו? (מה הרלוונטיות?)

### מה גורם לפוסט לעבוד
- וו שגורם לעצור את הגלילה
- פניה לקהל ספציפי (לא "כולם")
- ערך אמיתי (לא "buy my thing")
- CTA אחד ברור
- עקביות בוויזואל
