# קרוסלה - להפסיק להישמע מקצועית

## שקופית 1️⃣ - הוק (תופסת עין)

```
כשאתה מנסה להישמע מקצועית מדי
התוכן שלך נשמע בדיוק כמו כל אחד אחר
```

---

## שקופית 2️⃣ - הבעיה (סיפור אישי)

```
זה קרה לי גם

כתבתי פוסט "מקצועי"
אחרי שעה - אפס saves
אחרי שעה - אפס תגובות

רק אנשים שכבר מכירים אותי קראו אותו
```

---

## שקופית 3️⃣ - הסבר (למה זה קרה)

```
מה קרה?

כתבתי כמו ספר עסקי
כמו כל הספרים שקראתי
כמו כל המשווקות היפות

בדיוק כמו כולם

וזה היה הבעיה
```

---

## שקופית 4️⃣ - שינוי חשיבה (הרגע שהשתנה הכל)

```
המומנט שהשתנה הכל:

לקוח אחת שלי אמרה:
"אני שומרת את הפוסטים שלך 
כי אתה מדברת כמוני
לא כמו מנהלת עסקים"

וזה היה זה

התוכן שמישהו שומר 
זה לא 'טוב'
זה אמיתי
```

---

## שקופית 5️⃣ - הכלים (מה עשיתי)

```
מה עשיתי:

הפסקתי להישמע מקצועית
התחלתי לדבר כמו שאני
סיפרתי סיפורים אמיתיים
(גם הם שהיו קצת מביש)

השמירות?
טיפלו לעצמן
```

---

## שקופית 6️⃣ - CTA (מה אתה עושה עכשיו)

```
ועכשיו אתך:

בדקי את הפוסט האחרון שלך
שנראה לך "יותר מדי מקצועי"

תחליפי את שפת הספר העסקי
לשפה שלך

את הטקסט הירוק שאתה משתמשת בוואטסאפ

וראי מה קורה לשמירות

💬 וגלי לי בתגובה:
מה השתנה?
```

---

## כיתוב לפוסט

```
6 שקופיות שישנו את דרך החשיבה שלך על תוכן

כשאתה מנסה להישמע מקצועית מדי, התוכן שלך נשמע בדיוק כמו כל אחד אחר.

אם אתה מרגישה שהתוכן שלך "מקצועי מדי" ואנשים לא שומרים או משתפים:
👇 סקרול לסוף

💬 בתגובה: דוגמה מהתוכן שלך שנראה לך מדי מקצועי - גלי מה היה להוציא כדי להחזיר אותו לקול שלך
```

## המדריך לעריכה בקנווה

כל שקופית צריכה להיות **שונה**:

1. **שקופית 1** - וו חזק, ממוקד
2. **שקופית 2** - סיפור אישי (מה קרה לך)
3. **שקופית 3** - הסבר (למה זה קרה)
4. **שקופית 4** - הרגע שהשתנה (אהה!)
5. **שקופית 5** - פעולה (מה עשיתי)
6. **שקופית 6** - הזמנה (מה אתה עושה)

**זה סיפור שלם - לא 6 כפילויות של אותו דבר.**
